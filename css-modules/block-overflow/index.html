block-overflow

What is this?

テキストを切り捨て、max-linesによって設定された行数の後に省略記号またはカスタム文字列を挿入することによって、さらに多くのコンテンツが続くことを示す。

Status

2023年1月8日時点では、まだどのブラウザでも使えない上に、Mdnドキュメントも存在しない。

代替策として、CSS-Trick - Line Clampin’が紹介されている。
