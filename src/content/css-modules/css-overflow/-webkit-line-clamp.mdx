## {TITLE_LV2.bad}

### 上下の`padding`があると綺麗に見えない

`-webkit-line-clamp`を指定するとき、上下に`padding`があると、省略行の次の行が見えてしまう。

<MultiLineTextDemo style={frontmatter.styles.padding + frontmatter.styles.clamp} styleClass='has-padding-clamp' />

この場合、少なくとも`padding-bottom: 0;`が必要。

<MultiLineTextDemo style={frontmatter.styles.padding + frontmatter.styles.padding_bottom_reset + frontmatter.styles.clamp} styleClass='has-padding-clamp' />

### `-webkit-`群の組み合わせでしか機能しない

`-webkit-box`の正体は`flex`プロパティのベンダープレフィックス。

`flex`が草案時代に WebKit で`flex`を利用するときに使われていたプロパティだが、
`-webkit-box`の代わりに`flex`を指定すると`-webkit-line-clamp`は機能しない。

<MultiLineTextDemo style={frontmatter.styles.not_working} styleClass='not-working' />
